About the Project

Project Overview

This project aims to provide information about world population, the safest countries, and insurance companies in Canada through web scraping and data presentation.

Data Sources

The project collects data from the following sources:

  • Wikipedia: For world population and insurance company information.
  • World Population Review: For the list of safest countries.

Technologies Used

This project utilizes the following technologies:

  • Flask: A web framework used for creating the website and handling user requests.
  • SQLite: A lightweight and serverless relational database used to store the scraped data in three different tables.
